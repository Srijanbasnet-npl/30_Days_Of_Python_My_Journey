# 30 Days of Python My Journey

This repository documents my learning journey through Python over 30 days. It contains daily notes, examples, and small practice scripts that help me build a strong foundation in Python.

## Contents

- 01_Day_Introduction: Introduction to Python and a first "Hello, World!" program
- 02_Day_Variables: Learning about variables and basic data storage
- 03_Day_Operators: Exploring arithmetic and other operators

## How to Use

Each day folder contains Python files that can be run directly with:

```bash
python filename.py
```

## Goals

- Practice Python daily
- Build confidence with core concepts
- Keep learning notes in one place

Happy coding!
